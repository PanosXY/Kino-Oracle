# Kino-Oracle
A script that generates numbers for the Greek lottary game KINO based on the most drawn numbers

Installation
------------

Clone repository:
::

    git clone https://github.com/PanosXY/Kino-Oracle.git
    
Install the script:
::

    cd /path/to/repo
    sudo ./setup.sh
    
Usage
-----

After installing the script, you are ready to run it:
::

    kino.py
